# Marple Python Modules

A python package for modules shared across the Marple project.


### Install packages

`pip install git+ssh://git@github.com/marple-newsrobot/marple-py-modules.git`

### Run tests

`sh run test.sh`

### Update docs

`cd docs`
`make html`
